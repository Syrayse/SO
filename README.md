# Argus

Sistema capaz de controlo, monitorização e comunicação entre processos para obter uma ferramente para o eficiente processamento de tarefas concorrentes, com uma forte ênfase no armazenamento desses dados para futura utilização, por meios de uma arquitetura cliente - servidor.

# Utilização

De momento o *Argus* é capaz de lidar com as seguintes funcionalidades:

* Definir tempo máximo de inactividade de comunicação num pipe anónimo.
* Definir tempo máximo de execução de uma tarefa.
* Executar uma tarefa.
* Listar tarefas em execução.
* Terminar uma tarefa em execução.
* Listar registo histórico de tarefas terminadas.
* Apresentar ajuda à sua utilização.
* Consultar o *standard output* produzido por uma tarefa já executada.
* Alterar idioma do cliente.
